# Laravel Logic Simulator — KP Edition v2

A sophisticated, browser-based IDE designed to replicate the Laravel 11.x development experience. Practice Laravel architecture, routing, and logic directly in your browser without needing a local PHP environment.

## 🚀 Key Features

* **Virtual Filesystem**: Navigate a complete Laravel directory structure (Controllers, Models, Migrations, etc.).
* **Artisan Terminal**: Execute `php artisan` commands to dynamically generate files and run migrations.
* **Live Preview System**: Instantly render Blade templates and visualize route tables or database schemas.
* **Smart Code Editor**: Includes multi-tab support, syntax-specific snippets, and a built-in linter.
* **Global Search**: Powerful search overlay to find code strings across the entire virtual project.

## 🛠️ Getting Started

1.  **Download**: Save the `laravel-simulator-enhanced.html` file to your computer.
2.  **Run**: Open the file in any modern web browser (Chrome, Firefox, or Edge).
3.  **Code**: Start typing Laravel code in `web.php` or use the terminal to create new components.

## 🖥️ Built With

* **HTML5 & CSS3**: Custom modern UI with responsive layouts.
* **Vanilla JavaScript**: Core engine for the virtual filesystem and command interpretation.
* **Fira Code**: High-quality monospaced font for the professional coding experience.
